Version 1.0
一个给自习带来乐趣的应用
我在自习是一个免费的查找自习教室的应用，增加了找自习好友，帮占座，积分，签到排名等功能。为枯燥的校园自习生活带来乐趣。
  • 支持基于定位的自习教室查找
  • 签到，计划管理
  • 帮你推荐自习好友
  • 免费，无须注册
要求:
Android2.2+、GRPS、摄像
